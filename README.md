## About The Project
The Big Three Depository is a [bookdepository](http://web.archive.org/web/20230405210621/https://www.bookdepository.com/)-inspired e-commerce specialized in the works of "the big 3" of American science fiction. The objective of the project was to learn and get used to using react hooks, state propagation (prop-drilling & context) and react router.

## Project Duration
7 days with some fixes post-completion (see [commits](https://github.com/marcialcarrillo/konrad-project2/commits/master) after import).
